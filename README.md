# Projetos em C

Repositório principal dos meus projetos em C (com alguns projetos em C++ e MIPS Assembly), desenvolvidos ao longo da graduação em Ciência da Computação.

Mais do que uma lista de exercícios, este material mostra uma evolução técnica clara: de fundamentos de programação para estruturas de dados, métodos numéricos, autômatos, sistemas operacionais, concorrência, redes, arquitetura e grafos.

## Visão geral

- Total de projetos: **36**
- Período: **2023 -> 2026**
- Linguagens: **C (principal), C++, MIPS Assembly**
- Perfil: projetos acadêmicos com foco em implementação prática, análise de algoritmos e resolução de problemas reais

- Progressão técnica consistente em 4 anos

## Como executar os projetos

Cada pasta contém seu próprio contexto de execução (alguns com `Makefile`, outros com compilação direta).

### Opção 1: projetos com Makefile

```bash
make
./main
```

### Opção 2: compilação direta com GCC

```bash
gcc main.c -o main
./main
```

### Opção 3: quando há bibliotecas adicionais

- Projetos matemáticos podem exigir `-lm`
- Projetos com threads podem exigir `-lpthread`
- Projetos gráficos/OpenGL dependem de bibliotecas do sistema

Exemplo:

```bash
gcc main.c -o main -lm -lpthread
./main
```
